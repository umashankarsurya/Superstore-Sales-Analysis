# Superstore Sales Analysis

End-to-end retail sales analysis using Python — analysing 9,800 
sales records to surface revenue trends, regional performance 
gaps, and product insights.

## Tools Used
- Python (Google Colab)
- Pandas — data cleaning and analysis
- Matplotlib — data visualisation

## What This Project Covers
- Sales trend analysis (monthly, quarterly, yearly)
- Regional and city-level performance breakdown
- Product category and sub-category analysis
- Customer segment comparison
- Seasonal pattern identification
- Shipping behaviour analysis

## Key Findings
- Technology is the highest revenue category
- West region leads all regions in total sales
- Q4 consistently shows the strongest sales each year
- Small group of top customers drives majority of revenue

Superstore Sales Dataset — 9,800 rows, 18 columns
